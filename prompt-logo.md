Design a premium, modern logo for a software product named “Quebracho”, created by “Yacare Studio” from Chaco, Argentina.

Core concept:
- Fusion of nature + advanced AI tooling.
- Visual metaphor inspired by the quebracho tree (strength, resilience) and the yacare (regional identity), but in an abstract/minimal way.
- Must feel like a world-class developer tool brand (similar quality level to Linear, Notion, Cursor, Raycast).

Style:
- Minimal, geometric, clean, high-contrast.
- Flat vector look, no photorealism.
- Distinctive symbol + wordmark.
- Symbol should work as app icon (square), favicon, and terminal/avatar mark.

Color direction:
- Primary deep graphite: #1F2430
- Secondary slate: #323643
- Accent green: #4ADB94
- Optional warm earth accent (subtle): #B07A3F
- Provide versions for dark and light backgrounds.

Typography:
- Contemporary sans-serif, technical but friendly.
- Strong readability at small sizes.
- Wordmark text: “Quebracho”
- Optional subline for institutional use: “by Yacare Studio · Chaco, Argentina”

Deliverables:
1) Main horizontal lockup (symbol + wordmark)
2) Stacked lockup
3) Icon-only version
4) Monochrome version
5) Clear-space and minimum-size guidance
6) Export-ready SVG style

Creative constraints:
- Avoid clichés (no generic brain, chip, or robotic face).
- Avoid over-detailed animal/tree illustrations.
- Prioritize abstraction, memorability, and scalability.
- Must feel original, strong, and Latin American in soul but global in execution.
